# Telegram bot for exercises of Learn Python  

Bot can do next stuff:  

```
1. Счетчик слов - подсчет количества слов в веденном предложении  
2. Калькулятор - вычисление результата операции   
        Поддерживается три варианта калькулятора:  
          - Текстовый - вычисление результата по введенной строке (в строке могут быть цифры, символы арифметических операций и скобки)  
          - Кнопочный - управление с помощью кнопок  
          - Словесный - похож на текстовый за исключением того, что числа и операции в нем указываются словами (двадцать два плюс три)  
3. Планеты и созвездия - определение созвездия, в котором находится та или иная планета  
4. Полнолуние - получение информации о следующем полнолунии  
```
